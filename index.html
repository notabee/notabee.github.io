Ronit Jorvekar, Daniel Rodriguez-Criado, Luis J. Manso
Workshop of Physical Agents, 2020
project page / arXiv / video / code

OPAL-Net: A Generative Model for Part-based Object Layout Generation
Rishabh Baghel, Ravi Kiran Sarvadevabhatla,
arxiv, 2020
arXiv

We propose OPAL-Net, a novel hierarchical architecture for part-based layout generation of objects from multiple categories using a single unified model.
Video Surveillance for Violence Detection Using Deep Learning
Rishabh Baghel, Manan Sharma
Advances in Data Science and Management, 2020

In order to detect violence through surveillance cameras, we provide a neural architecture which can sense violence and can be a measure to prevent any chaos
